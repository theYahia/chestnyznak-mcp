# /verify-product — Проверка маркировки

## Алгоритм
1. Вызови `check_marking_code` с кодом
2. Покажи: подлинность, производитель, название товара

## Формат ответа
```
## Проверка маркировки
**Товар**: [название]
**Производитель**: [название]
**Статус**: подлинный / не найден
```

## Примеры
```
/verify-product 010460406000600321CPGpRgR
```
