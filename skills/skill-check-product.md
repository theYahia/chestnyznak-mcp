# Проверка маркировки товара

Используй инструмент `check_marking_code` для быстрой проверки подлинности по коду маркировки.
Если пользователь хочет подробную информацию (производитель, владелец, группа товаров), используй `get_product_info`.

## Шаги

1. Запроси код маркировки у пользователя, если не указан.
2. Вызови `check_marking_code` с кодом (по умолчанию `codeType: datamatrix`; для QR/штрихкода укажи `qr`/`ean13`).
3. Если товар найден и пользователь хочет подробности — вызови `get_product_info`.
4. Сообщи результат: найден (`found`), проверка пройдена (`valid`), статус, название товара.

## Примеры запросов

- "Проверь код маркировки 0104600702028445"
- "Этот товар настоящий? Код: 010460070202844521"
- "Проверь маркировку на молоке"
