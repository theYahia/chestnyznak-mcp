# Информация о товаре по GTIN

Используй инструмент `search_products` для получения информации о товаре по его GTIN
из True API. Требуется установленный `CHESTNYZNAK_TOKEN`.

> Поиск по названию/бренду — это Национальный каталог (nk.crpt.ru), отдельный API,
> здесь не поддерживается. Этот инструмент работает по GTIN.

## Шаги

1. Запроси GTIN у пользователя, если не указан.
2. Вызови `search_products` с GTIN.
3. Если токен не задан — инструмент вернёт ошибку; объясни, как получить и установить
   `CHESTNYZNAK_TOKEN` (см. README → «Авторизация»).
4. Покажи результаты: название, бренд, производитель.

## Примеры запросов

- "Информация о товаре с GTIN 04600702028445"
- "Что за товар с GTIN 04607177079955"
