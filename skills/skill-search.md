# Поиск товаров по названию

Используй инструмент `search_products` для поиска товаров по названию, бренду или GTIN.
Требуется установленный CHESTNYZNAK_TOKEN.

## Шаги

1. Запроси поисковый запрос у пользователя, если не указан.
2. Вызови `search_products` с запросом.
3. Если токен не задан — объясни, как установить CHESTNYZNAK_TOKEN.
4. Покажи результаты: название, бренд, производитель.

## Примеры запросов

- "Найди молоко Простоквашино"
- "Поищи товары бренда Danone"
- "Найди товар по GTIN 4600702028445"
